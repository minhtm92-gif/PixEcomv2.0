⚠️ How to capture requestId + Status Code

All errors MUST include requestId + status code. No requestId = cannot debug.

1 On UI: When error occurs, the toast/alert shows "Request ID: xxxxxxxx". Copy it.
2 DevTools: Press F12 → Network tab → redo the failing action → find red request → Headers tab → look for Status Code + x-request-id
3 Console: Press F12 → Console tab → find error line with requestId → copy
